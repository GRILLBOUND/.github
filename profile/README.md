## 🎯 What We're Building

**GRILLBOUND** is an open-source multiplayer horror game set in a burger kiosk during the night shift. Work together to complete orders, manage stations, and survive until dawn—all while evading a relentless hunter.

Think *Five Nights at Freddy's* meets *Overcooked* with co-op multiplayer.
